# Product: Expense & Budget Visualizer

A single-page, client-side web app for tracking personal expenses. Users can add transactions (name, amount, category), view a running total balance, and see spending broken down by category in a pie chart.

## Core Capabilities
- Add and delete transactions
- Organize transactions by custom or default categories (Food, Transport, Fun)
- Filter transactions by month
- Visualize spending per category via a Chart.js pie chart
- Persist all data in `localStorage` (no backend)
- Light/dark theme toggle, persisted across sessions

## Target User
Individual users tracking personal day-to-day spending directly in the browser — no login, no server required.
